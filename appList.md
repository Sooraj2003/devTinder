# Api Endpoints

 ## authRouter 
   - POST /signup
   - POST /login
   - PPOST /logout


## profileRouter
   - GET /profile/view
   - PATCH /profile/edit


## requestRouter
   - POST /request/send/:status/toUserId
   - POST /request/view/:status/requestId

## userRouter
   - GET /user/requests
   - GET /user/connections
   - GET /user/feed
